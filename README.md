# Students_Lottery_App
Lottery Draw System
A simple Python application for conducting random lottery draws with participant management.
Features

Add participants with name, address, and phone number
View all participants in a list
Delete individual participants
Clear all data
Random lottery draw with animation
Data persistence using JSON

Requirements

Python 3.6+
tkinter (included with Python)


How It Works

Add participants by filling in the form
View all participants in the list
Click "Draw Lottery" to randomly select a winner
Data is automatically saved to students.json
